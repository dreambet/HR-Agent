# 打招呼功能完整调试总结

## 日期
2026-05-19

## 功能概述
智联招聘主动打招呼功能：自动向候选人发送打招呼消息，包括选择相关职位和填写招呼语。

## 打招呼流程

### 完整流程步骤
1. 访问智联招聘搜索页面
2. 输入岗位关键词搜索
3. 找到目标候选人并点击"打招呼"按钮
4. 在"选择沟通职位"对话框中选择相关职位
5. 点击"确定"按钮
6. 在"AI招呼语上线了"对话框中：
   - 切换到"自己设置招呼语"
   - 填写招呼语
   - 点击"使用并发送"

### 两个对话框说明
| 对话框名称 | 功能 | 关键操作 |
|-----------|------|---------|
| 选择沟通职位 | 选择要发送的职位 | 在搜索框输入职位关键词过滤，选择后点"确定" |
| AI招呼语上线了 | 填写招呼语 | 切换到"自己设置招呼语"，填写后点"使用并发送" |

## 招呼语规则

### 招呼语格式（已确认）
```
您好，我对您的简历很感兴趣，希望能有机会沟通。我司正在招聘【职位名称】职位，期待与您进一步沟通。
```

### 示例
- 搜索关键词：软件开发工程师
- 匹配职位：资讯工程师
- 发送招呼语：您好，我对您的简历很感兴趣，希望能有机会沟通。我司正在招聘【资讯工程师】职位，期待与您进一步沟通。

### 招呼语类型
- 使用"自己设置招呼语"（不是AI个性化招呼语）

## 职位匹配规则

### 匹配逻辑
1. 从企业职位列表API获取所有现有职位
2. 根据搜索关键词与职位进行语义匹配
3. 优先匹配同一领域的职位

### 职位分类
| 分类 | 关键词 |
|------|--------|
| IT相关 | 软件、开发、程序员、工程师、技术、IT、资讯、网络、系统、数据、数据库、前端、后端、全栈、架构 |
| 制造业 | CNC、加工、车床、铣床、模具、钳工、技工、操作工、工厂 |
| 管理 | 主管、经理、总监、负责人、组长 |
| 人事 | 招聘、HR、人事、人力资源、培训、绩效 |

### 匹配分数规则
| 匹配类型 | 分数 |
|---------|------|
| 完全包含关键词 | 100 |
| 关键词包含职位名 | 90 |
| 去除特殊字符后匹配 | 80/70 |
| 同一分组（语义相关） | 75-85 |
| IT相关关键词匹配到非IT职位 | 20 |
| 部分字符匹配 | 50 |

## 关键问题解决

### 问题1：Cookie失效
- **现象**：`{"code":401,"message":"您的账号已在其他设备或浏览器登录"}`
- **解决**：用户提供新Cookie并更新到 `/root/.openclaw/workspace-HR-Agent/config/zhaopin_cookies.txt`

### 问题2：下拉列表只显示部分选项
- **现象**：下拉列表只显示6个选项，但实际有41个职位
- **原因**：下拉列表是虚拟化渲染，只渲染可见区域的选项
- **解决**：在下拉列表的搜索输入框中输入职位关键词进行过滤

### 问题3：按钮点击被拦截
- **现象**：`element intercepts pointer events`
- **原因**：有其他元素覆盖在按钮上
- **解决**：使用JavaScript直接调用`click()`方法绕过拦截

### 问题4：候选人匹配错误
- **现象**：打招呼发送给错误的人
- **解决**：使用多字段精确匹配（姓名+年龄+工作年限+学历）
