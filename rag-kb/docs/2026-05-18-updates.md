
## 详细简历报告模板格式（2026-05-18）

### 模板文件
`/lhcos-datas/reports/详细报告/简历分析报告样例.md`

### 核心格式要求

#### 1. 工作经历模块
- 格式：时间+公司+职位，下方一行总结
- 不是罗列bullet points，而是用逗号连接成一句连贯总结
- 示例：
```
**2022.08-至今**：东大金智科技 - it运维工程师
负责监控IPS和IDS告警信息，排查Shiro、FastJson等组件漏洞利用尝试...，排查服务器日志，分析弱口令爆破...，在同一个网段下部署多个蜜罐...
```

#### 2. 项目经历模块
- 格式：项目名称（时间），下方一行总结
- 同样用逗号连接成连贯段落
- 示例：
```
**云网络架构升级**（2025.03 - 2026.03 (1年)）：负责网络优化，通过 BGP 路由策略调优，成功解决核心业务跨 IDC 网络延迟问题；通过精细化防火墙 ACL 策略联动 IDS/IPS 系统
```

#### 3. 教育背景模块
- 格式：学校：专业 学历 时间
- 示例：
```
**郑州科技学院**：计算机科学与技术 本科 2018.09 - 2022.06
```

### 总结式呈现原则
- 工作经历：提取3个关键要点，用逗号连接成连贯段落
- 项目经历：提取2个关键要点，同样用逗号连接
- 不要简单罗列所有bullet points，要生成连贯的文字段落
- 去除"主要负责"前缀（因为内容本身已经说明）
