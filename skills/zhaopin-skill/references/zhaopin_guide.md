# 智联招聘网站爬取参考

## 网站结构

### 主要URL

- **智联招聘首页**: https://www.zhaopin.com
- **简历搜索页**: https://sou.zhaopin.com
- **个人简历中心**: https://i.zhaopin.com
- **企业招聘后台**: https://ehire.zhaopin.com

### 搜索参数说明

```
https://sou.zhaopin.com/?jl={location}&kw={keywords}&kt=3
```

| 参数 | 说明 | 示例 |
|------|------|------|
| jl | 工作地点 | 北京、上海、深圳 |
| kw | 关键词 | Python开发工程师 |
| kt | 简历类型 | 3=全部 |

## Cookie 获取方法

1. 登录智联招聘企业版 (https://ehire.zhaopin.com)
2. 按 F12 打开开发者工具
3. 切换到 Network（网络）标签
4. 刷新页面
5. 点击任意请求，在 Headers 中找到 Cookie 字段
6. 复制完整的 Cookie 字符串

## 页面选择器参考

### 简历列表选择器

```css
.joblist-box          /* 简历列表项 */
.resume-name          /* 简历姓名 */
.resume-title         /* 期望职位 */
.resume-info          /* 基本信息 */
```

### 简历详情页选择器

```css
.basic-info           /* 基本信息区域 */
.work-experience      /* 工作经历区域 */
.project-experience   /* 项目经历区域 */
.education-info       /* 教育背景区域 */
.skill-info           /* 技能证书区域 */
```

## 注意事项

1. **Cookie 时效**: 智联 Cookie 通常有时效限制，发现认证失败时请重新获取
2. **请求频率**: 请合理控制请求间隔（建议 2-3 秒），避免被封禁
3. **数据隐私**: 所有获取的候选人信息仅供内部招聘使用，严禁外泄
4. **反爬机制**: 如遇验证码或反爬提示，可能需要切换为账号密码登录
